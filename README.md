# AI Resume Screening System

This project analyzes resumes and matches skills with job requirements using Python and Flask.

## Features
- Resume PDF upload
- Skill extraction
- Match percentage calculation
- Web interface using Flask

## Technologies Used
- Python
- Flask
- HTML
- CSS

## How to Run

1. Install requirements

pip install flask pdfplumber

2. Run application

python app.py

3. Open browser

http://127.0.0.1:5000
